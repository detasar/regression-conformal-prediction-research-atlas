# Security

## Supported Surface

Security review for this repository covers the Research Atlas package: static HTML, aggregate CSV/JSON artifacts, Python smoke-test code, package metadata, and documentation.

## Reporting

Please report security concerns privately to Emre Tasar at <detasar@gmail.com>. Include the affected path, a short reproduction note, and whether the issue may expose credentials, raw data, restricted ledgers, or nonredistributable source material.

## Data Handling

The repository should not contain credentials, secrets, raw restricted ledgers, local caches, or nonredistributable datasets. Public artifacts use manifest-plus-summary traceability for restricted sources.
